## How Atmet Works

<Steps>
  <Step title="Connect your apps">
    Sign in and link the tools your team already uses — Gmail, Google Drive, Telegram, GitHub — with one-click OAuth or an API key.
  </Step>
  <Step title="Describe the workflow">
    Open AI Core and tell the agent what you want in plain language. It will ask follow-up questions through smart input panels and draft the workflow for your review.
  </Step>
  <Step title="Approve and activate">
    Review the workflow steps, confirm any sensitive actions in plain language, then activate. Atmet runs the workflow automatically on every trigger — no manual input needed.
  </Step>
  <Step title="Monitor and refine">
    Track every run from the Workflows dashboard. See exactly which steps ran, what was returned, and adjust the workflow anytime via chat.
  </Step>
</Steps>
